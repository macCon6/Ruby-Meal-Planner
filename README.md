# Ruby-Meal-Planner

A Ruby command-line application that creates weekly dinner plans based on budget, dietary restrictions, and taste preferences. 

This planner generates meals from a pre-defined set of meals whose total weekly cost stays within a specified budget.

## Features

- Repeated attempts to create a budget-friendly meal plan
- Vegetarian filtering
- Object-oriented Ruby design
